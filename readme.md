# ROS2实现kitti数据可视化

  本仓库代码实现了使用ROS2进行kitti的数据可视化

## 测试环境

- 系统：Garuda（宿主机）+Ubuntu 24.04LTS（distrobox）

- ROS2版本：jazzy

- Python版本：3.12.8
